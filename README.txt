List of written methods for the DAHashTable (Direct Address Hash Table) class:
    insert:
        In: An int 
        Out: Nothing
        Effect: Inserts the passed int value into the table
    delete:
        In: An int
        Out: Nothing
        Effect: Deletes the value at the passed key
    search:
        In: An int
        Out: An int
        Effect: Takes in a key value and returns the value at that location
    key:
        In: An int
        Out: An int
        Effect: Creates a key out of a passed int value

List of written methods for the Node class:
    getKey:
        In: Nothing
        Out: An int
        Effect: Returns the stored key value
    getValue: 
        in: Nothing
        Out: An int
        Effect: Returns the stored 'value' value
